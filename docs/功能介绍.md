# 功能介绍

## 一、系统概述

AI 作业批改系统是一个基于人工智能的教育辅助平台，支持教师创建作业、学生在线提交、AI 自动批改与反馈分析。系统覆盖客观题（选择/填空）的模板匹配评分和主观题的 AI 大模型语义批改，适配英语、数学等多学科场景。

---

## 二、用户角色

### 教师

- 创建和管理作业
- 设置客观题（选择题 / 填空题）和主观题
- 查看学生提交情况
- 触发 AI 批改
- 查看班级统计数据

### 学生

- 浏览可用作业列表
- 在线作答（客观题 + 主观题）
- 拍照上传手写作业（OCR 识别）
- 查看 AI 批改结果与反馈

---

## 三、核心功能

### 3.1 作业管理

| 功能       | 说明                                 |
| ---------- | ------------------------------------ |
| 创建作业   | 设置标题、科目、描述、参考答案、满分 |
| 客观题编辑 | 添加选择题（ABCD 四选一）和填空题    |
| 主观题描述 | 填写作业要求，供 AI 参考批改         |
| 作业列表   | 按科目查看，显示提交数和批改状态     |

### 3.2 作业提交

| 功能       | 说明                                 |
| ---------- | ------------------------------------ |
| 客观题作答 | 点选选择题选项 / 填写填空题答案      |
| 主观题作答 | 文本框输入答案                       |
| 图片 OCR   | 上传手写作业照片，自动识别文字并填入 |
| 重新提交   | 支持修改答案后再次提交               |

### 3.3 AI 智能批改

#### 客观题 — 模板匹配

- 将学生答案与标准答案逐题比对
- 计算正确率并给出逐题对错详情
- 无需 AI API，即时返回结果

#### 主观题 — 大模型批改

- 基于学科专用 Prompt 生成评分
- 返回：得分 / 总体评价 / 优点 / 不足 / 改进建议
- 支持 4 个 AI 提供商（Gemini、DeepSeek、SiliconFlow、OpenAI）

#### 混合题型

- 客观题和主观题按 40:60 加权计算总分
- 合并两部分的反馈信息

### 3.4 学科专用评价

系统为 8 个学科定制了差异化 AI 提示词：

| 学科   | 评价重点                         |
| ------ | -------------------------------- |
| 数学   | 计算过程、公式运用、逻辑推导     |
| 语文   | 文章结构、修辞手法、立意深度     |
| 英语   | 语法正确性、词汇、句式多样性     |
| 物理   | 概念理解、公式选用、单位换算     |
| 化学   | 方程式配平、反应条件、实验规范   |
| 生物   | 概念准确性、实验设计、图表分析   |
| 历史   | 史实准确性、因果分析、论述逻辑   |
| 计算机 | 代码逻辑、算法正确性、复杂度分析 |

### 3.5 图文识别（OCR）

- 基于 Tesseract.js，完全免费
- 支持中文 + 英文混合识别
- 学生上传照片 → 文字提取 → 自动填入答案区 → 可编辑后提交
- 返回识别置信度百分比

### 3.6 数据统计

| 指标     | 说明                   |
| -------- | ---------------------- |
| 作业总数 | 教师创建的作业数量     |
| 提交总数 | 收到的学生提交数量     |
| 已批改   | 完成 AI 批改的提交数   |
| 平均分   | 全部批改结果的平均得分 |

## 四、多端适配

| 平台       | 支持情况                          |
| ---------- | --------------------------------- |
| 桌面浏览器 | ✅ 完整侧栏 + 大屏布局            |
| 平板       | ✅ 紧凑侧栏 + 自适应网格          |
| 手机浏览器 | ✅ 抽屉菜单 + 底部导航 + 触摸优化 |

移动端特性：

- ☰ 汉堡菜单替代固定侧栏
- 底部导航栏快速切换
- 44px 最小触摸热区
- 表单输入 16px 字号（防 iOS 自动缩放）
- 适配 iPhone 安全区域

## 五、安全特性

| 特性         | 实现方式                 |
| ------------ | ------------------------ |
| 密码加密     | bcryptjs 单向哈希        |
| 身份认证     | JWT Token                |
| 角色控制     | 教师 / 学生权限分离      |
| 文件上传限制 | 10MB 大小 + 图片格式校验 |
